# Foreca - Digitalisasi Strategi UMKM 🚀

Foreca adalah sistem pendukung keputusan berbasis web yang dirancang khusus untuk mendigitalisasi strategi penjualan UMKM. Memanfaatkan *data analytics* dan model *deep learning*, sistem ini mengolah data transaksi menjadi prediksi penjualan 7 hari ke depan, *insight* tren, dan rekomendasi tindakan otomatis yang interaktif.

---

## 🎯 Fitur Utama

- **Prediksi Penjualan (7 Hari Ke Depan):** Memanfaatkan model *machine learning* / *deep learning* (CNN-LSTM) untuk melakukan *forecasting* penjualan secara akurat berdasarkan riwayat data transaksi.
- **Insight Tren Penjualan:** Visualisasi data interaktif menggunakan Streamlit untuk memonitor pergerakan tren transaksi dari waktu ke waktu.
- **Analisis Penyebab:** Menganalisis faktor penyebab perubahan performa penjualan secara spesifik dari sisi produk maupun rentang waktu tertentu.
- **Rekomendasi Tindakan (Actionable):** Menghasilkan rekomendasi strategi bisnis berbasis fakta (*data-driven*) dengan bantuan Generative AI (LLM) untuk mendukung pengambilan keputusan operasional.
